css-forgettable

Posted on 2019-06-22 | In css

A summary for forgettable css styles

  • Box-shadow
    box-shadow: (inset) offset-x offset-y blur-radius spread-radius color

example:

    box-shadow: 10px 30px 4px 20px rgba(0, 0, 0, 0.6);
  • filter
    filter: blur(9px);                      // 模
    filter: grayscale(1);                   // 灰度
    filter: brightness(2.3);                // 亮度
    filter: contrast(4.4);                  // 对比度
    filter: saturate(3.6);                  // 饱和度
    filter: hue-rotate(185deg);             // 色相旋转
    filter: invert(1);                      // 反
    filter: drop-shadow(0px 0px 5px #000);  // 阴影
    filter: opacity(55%);                   // 透明度
    filter: sepia(0.77);                    // 褐色

drop-shadow 与 box-shadow 不同，只给不透明的地方加阴影
